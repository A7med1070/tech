# دليل رفع واستضافة موقع فريش إير للمكيفات

تاريخ الأسعار والمراجعة: 2026-05-02.

## ما هو نوع المشروع؟

المشروع موقع Static HTML/CSS/JS، يعني لا يحتاج قاعدة بيانات ولا Backend حتى يعمل. يمكن رفعه مباشرة على أي استضافة Static.

الملفات المهمة:

- `index.html`: الصفحة الرئيسية.
- `about.html`: من نحن.
- `services.html`: الخدمات.
- `products.html`: المنتجات.
- `product.html`: تفاصيل المنتج.
- `gallery.html`: المعرض.
- `reviews.html`: التقييمات.
- `contact.html`: التواصل.
- `styles.css`: التصميم والـ responsive.
- `app.js` و `content.js`: الحركة، القائمة، الفلاتر، السلايدر، اللايت بوكس.
- `assets/logo.svg`: شعار فريش إير عالي الدقة بصيغة SVG.
- `assets/img`: الصور المحلية.

## طريقة الرفع الأسرع

ارفع كل محتويات فولدر المشروع كما هي. اجعل `index.html` في جذر الاستضافة مباشرة.

لا ترفع الملف المضغوط نفسه إلى جذر الموقع إلا لو منصة الاستضافة تطلب ZIP upload.

## تشغيل محلي

لو عندك Node.js:

```bash
npm run dev
```

ثم افتح:

```text
http://localhost:4173
```

ويمكن أيضا فتح `index.html` مباشرة من المتصفح لأن المشروع Static.

## التكلفة المتوقعة

### أرخص خيار عملي لشركة

Cloudflare Pages أو Netlify Free:

- الاستضافة: 0 دولار شهريا.
- SSL: مجاني.
- CDN: مجاني.
- دومين مخصص: اختياري.

لو بدون دومين مخصص: التكلفة 0 دولار.

لو مع دومين `.com` من Namecheap حسب السعر الظاهر وقت المراجعة:

- أول سنة: 11.28 دولار.
- التجديد السنوي: 18.48 دولار.

إجمالي أول سنة مع استضافة مجانية ودومين `.com`: 11.28 دولار فقط.

### خيار Netlify مدفوع

- Free: 0 دولار.
- Personal: 9 دولار شهريا.
- Pro: 20 دولار شهريا.

إجمالي أول سنة Netlify Personal + دومين `.com`: 108 + 11.28 = 119.28 دولار.

### خيار Vercel

Vercel Hobby مجاني لكنه مخصص للمشاريع الشخصية وغير التجارية، لذلك لا أنصح به لموقع شركة. الخيار التجاري المناسب:

- Vercel Pro: 20 دولار لكل مستخدم شهريا.

إجمالي أول سنة Vercel Pro لمستخدم واحد + دومين `.com`: 240 + 11.28 = 251.28 دولار.

### دومين سعودي `.sa`

تسجيل الدومين السعودي يتم عبر SaudiNIC والجهات المسجلة المعتمدة. صفحة CST تذكر أن إدارة التسجيل تتم عبر مسجلين معتمدين، لذلك السعر النهائي يعتمد على المسجل الذي ستختاره.
